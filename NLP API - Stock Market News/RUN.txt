Quick Start

- Install dependencies:
  - python -m pip install -r stock_sentiment_nlp/requirements.txt
- Start the server (recommended):
  - python server.py
- Open the app:
  - http://127.0.0.1:5000/
Alternative Start

- Using Flask CLI:
  - python -m flask --app stock_sentiment_nlp.app run --host 127.0.0.1 --port 5000
Set Your Finnhub Key

- In the UI, paste your key in the “Enter Finnhub API key” field and click Save Key.
- Or via environment variable before running:
  - Windows PowerShell: $env:FINNHUB_API_KEY="YOUR_KEY"
Use The App

- Click Test API to confirm connectivity and sample data.
- Choose a symbol (e.g., AAPL) and click Analyze to render charts and the table.
Files of Interest

- Entrypoint: server.py
- Requirements: requirements.txt
- App: app.py
Troubleshooting

- Port in use: change the port, e.g., python -m flask --app stock_sentiment_nlp.app run --port 5001
- Nothing shows on first load: hard‑refresh (Ctrl+F5) to ensure the latest JS is loaded.
- No Finnhub key: app still works with sample news; add a key for live data.
- Verify without running a server:
  - python -m stock_sentiment_nlp._route_test
  - python -m stock_sentiment_nlp._verify
